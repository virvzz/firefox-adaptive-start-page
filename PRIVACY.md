# Adaptive Start Page Privacy Policy

Adaptive Start Page is a local Firefox new tab extension. It does not use
analytics, advertising, telemetry, tracking pixels, or a developer-operated
server.

## Data stored locally

The extension stores its settings, tiles, folders, theme choices, imported
profile data, and user-added images/icons in Firefox local extension storage or
IndexedDB on your device.

Bookmark-based features read Firefox bookmarks so the extension can show and
manage bookmark tiles and folders. Open-tab and recently-closed-tab features read
Firefox tab/session data so the local search and quick-access widgets can work.
This data is not sent to the developer.

## Optional external requests

By default, external page previews, site icons, and weather are turned off.

If you enable "Online thumbnails and site icons", the URLs or domains of tiles
and bookmark tiles are requested from third-party preview services so images can
be displayed:

- WordPress mShots
- Google Favicons
- thum.io

If you enable the weather widget, the configured weather location is requested
from wttr.in. If the location field is empty, wttr.in may infer a location from
the network request.

You can turn these features off at any time in the extension settings. When they
are off, tiles use local letter/color previews and no tile URLs are requested
from preview services.

## Data sharing

The developer does not sell, rent, or share user data. The extension has no
developer-operated backend. Optional third-party requests are made directly by
Firefox when the user enables the corresponding feature.
